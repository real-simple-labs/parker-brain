# Process — Build Brand Script-Voice Profile

Build a brand's voice fingerprint from its own winning scripts before writing or adapting anything. This is the operational version of the profile method in `creative-strategy-context/spoken-script-voice.md`. The profile becomes the brand baseline every script is checked against, so generated lines land in the brand's register instead of generic ad-voice.

This is step one of every scripting request. Run it before find-reference-ads.

## Required inputs

- The active brand and its ad account access via Parker MCP.
- The brand's vault, for captured competitor and inspo scripts if the brand's own account is thin.
- The niche, so the right organic creators can be pulled if needed.

## When to pick

- **Always run first.** Every script request starts here. The profile is the brand baseline. Without it, adaptation drifts to generic ad-voice.
- If a current profile already exists for the brand, load it and refresh it against the latest winners rather than rebuilding from scratch.

## The visual twin

When the brand's visual-vocabulary doc exists at `sub-context-docs/visual-vocabulary.md`, load it with the voice profile. The two are twins. Voice is what the brand sounds like; the vocabulary is what it looks like — the brand's filmed settings, talent situations, and product moves, classified in-play, adjacent, and out-of-play per the method at `global/knowledge/creative-strategy/visual-vocabulary-method.md`. The voice profile anchors the words a script writes; the vocabulary anchors the visual direction each beat carries. Load both so the script's sound and look both belong to the brand.

## Source order

Pull in this order. Stop when the fingerprint is clear.

1. **The brand's own account scripts, first.** Top spenders and top hook-rate video units, with full transcripts. This is the brand's actual voice and always the primary source.
2. **Captured competitor and inspo scripts in the brand's vault, next.** Use when the brand's own account is thin or too polished to hear a voice from.
3. **The niche's top organic creators, last.** The purest spoken register — how the audience actually talks about the category. Use to ground diction and address when paid scripts are too produced.

## Execution

### Step 1 — Pull the scripts
Pull 10-15 of the brand's own winning video scripts with full transcripts. Sort by spend and by hook rate; the units that win on both are the truest signal. If the brand's account is thin, add vault competitor scripts, then niche organic.

### Step 2 — Read every one out loud
The fingerprint is audible, not visible. Read each script at performance speed. You are listening for what recurs — a move that shows up across multiple winners is the brand's signature, not a one-off.

### Step 3 — Note the five fingerprint dimensions
- **Pacing.** Sentence length, how much it jumps, how fast the brand cuts between thoughts.
- **Diction.** The brand's actual words and exact recurring phrases. Pull them verbatim.
- **Signature moves.** The structural tics the brand reuses — personification, spec lists, body-showing, admitting the catch.
- **Direct-address habit.** Who the brand talks to and how — crudely about the body, or as a peer sharing a tip.
- **CTA shape.** Exactly how the brand closes — its risk-reversal phrasing, its sign-off, its URL read.

### Step 4 — Build the banned-by-absence list
List the words and constructions that never appear in the brand's winners. Their absence is part of the voice. A word no rule forbids but the brand never uses is banned-by-absence, and using it would break the register. Treat this list as a hard constraint on the adaptation.

### Step 5 — Run the spoken-register read
Check the brand's winners against the AI-tells list in `spoken-script-voice.md` — confirm where the brand naturally avoids em-dash cadence, triplets, "it's not just X it's Y," signposted transitions, and over-clean grammar. The brand's natural avoidances become rules the adaptation must hold.

## Output content

- The voice fingerprint: pacing read, diction list with exact recurring phrases, signature moves, direct-address habit, CTA shape.
- The banned-by-absence list.
- The register the brand sits in — comedy, confession/testimony, authority, organic-creator, or a named blend.
- The source tier used — own account, vault competitor, or niche organic — and why.
- The exact verbatim phrases the adaptation should reuse as anchors.

## What never to do

- Skip straight to find-reference-ads without building the profile. The reference is adapted *into* this profile.
- Profile from a competitor's account instead of the brand's own. The brand's account is always the primary source.
- Read the scripts silently. The fingerprint is audible — read them out loud.
- Sand off the brand's mess — its "so," "okay," repeated words, exact numbers. Those are the fingerprint.
- Treat the banned-by-absence list as optional. It is a hard constraint.
- Invent recurring phrases the brand does not actually use. Every anchor phrase is verbatim from the corpus.
