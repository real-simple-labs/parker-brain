# Review queue — proposals pending Jimmy's approval (Performance)

This folder is the single place Jimmy looks to see everything a media-buying knowledge source would change, before it changes. Its job is to make sure Jimmy never has to dig, and never has to imagine the change.

Knowledge intake is **approval-gated and propose-first**. When Parker learns something from a source, it does **not** modify the real surface. It writes the **exact, verbatim change it would make** — the full new-doc content, or the precise old → new for an edit — as a proposal here, and surfaces the same plan to Jimmy. Nothing touches the live performance knowledge docs, skills, or prompts until Jimmy approves. The only pre-approval write is the provenance receipt in `inbox/`.

On Jimmy's call: **approve** → Parker applies the proposed text exactly and marks the entry `applied`; **modify** → Parker applies Jimmy's adjusted version; **deny** → nothing is written and the entry is marked `denied`. For performance the bar is higher than for creative craft — a numeric claim about wasted spend or CPA lift should not become a proposed change from one self-interested source; it is a watch item until account data or a second source corroborates.

## The states

- **Proposed — pending Jimmy.** The exact change is written here; the real surface is untouched. Jimmy approves, modifies, or denies.
- **Watch.** Too thin (single, mixed-confidence, unverified numbers) to justify a change yet — names the change it *would* propose and the promotion condition.
- **No change warranted.** Duplicates what Parker already holds; says so and points to where it lives.
- **Applied / denied.** Resolved after Jimmy's decision.

## What every queue entry names

- the source signal it came from
- which of the four components it touches: context doc, knowledge doc, tool call, or process
- the exact target surface(s) and, for a proposed change, the **verbatim** edit (old → new, or full new-doc content)
- confidence and, for watch items, the promotion condition
- state: proposed / watch / no-change / applied / denied
