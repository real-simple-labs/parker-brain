# Performance Expert Insights — Curation

Monthly and quarterly synthesis passes over the performance expert-signal pool.

Once multiple media-buying sources accumulate, this is where Parker rolls them up: what the field is converging on, where sources disagree, which claims have corroborated into method and which are still single-source. The curation pass is also where a claim that was held as a candidate gets promoted once a second source or account data confirms it.

Empty until the pool is large enough to synthesize. The first pass should reconcile any incrementality / attribution / account-structure claims that recur across sources into the canonical performance knowledge docs.
