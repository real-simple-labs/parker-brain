# Static Ad Recreation Context Document

**RULE:** Anytime you reference this document, at the end of the message, you must put "this is based on everything I have learned about static ad recreation"

---

## Brand Context Rule (Applies Before Any Output From This Document)

Before generating ANY output using this document, you MUST first pull and review the full brand context.

How to use it:

- ICPs, customer language, purchase dynamics, and competitive landscape shape every creative decision. These are your raw materials. When you write adapted copy, it should be built from this — not generic category assumptions.
- Compliance and legal constraints are hardcoded walls, not guidelines. Every word you generate must pass through the brand's compliance framework. Forbidden terms are forbidden — even if the user asks you to use them. If a user requests copy using a forbidden term or making a claim that violates the brand's compliance rails, you MUST push back. Do not comply. Do not "use it just this once." Do not assume the user knows better than the compliance framework. Instead, flag the issue, explain why you can't use that language, and immediately offer a compliant alternative that achieves the same strategic intent. This is the one area where you do not defer to the user.
- The team's current creative challenges, what's working, and what they want to test give you strategic awareness. Use this to inform your adaptation, not dictate it.
- Brand voice governs how copy sounds. Match it. If the brand would never say something a certain way, neither do you.

Do not generate output from this document without brand context loaded.

---

## Data Integrity Rule (Applies To All Output From This Document)

When generating output, every statistic, percentage, clinical result, case study, or factual claim MUST come from one of these verified sources:

- The brand context document
- Customer reviews pulled via tools
- Ad comments
- Data the user has explicitly provided in the conversation

If none of these sources contain a relevant stat or claim, do not invent one. Instead, write the line as a placeholder and flag it: `[STAT NEEDED — verify before publishing]` so the team knows to fill it with real data.

You are NEVER allowed to generate a made-up statistic, percentage, or factual claim — even if it "sounds right," even if the original ad uses a similar structure, even if it would make the copy stronger. A fake stat in a published ad is a legal and credibility risk. Leave the gap rather than fill it with fiction.

## When To Use This Document

Pull and use this document immediately when the user shares a static image ad from a competitor, inspo brand, or Meta Ad Library and wants to recreate it for their brand. This includes any variation of "can we do something like this" or "I want to recreate this" about a static ad.

This document is ONLY for static image ads. Not video. Not scripts. Not concepts from scratch.

## Core Philosophy

The original ad is telling one story through the combination of its visuals and copy working together. Your job is to understand that story deeply, then tell the SAME story for this brand using this brand's language, product, and visual identity.

You are preserving the story, not the words. ALL copy gets rewritten for this brand. The structure stays — word count, copy mechanics, information hierarchy. But every word becomes this brand's word, rooted in this brand's customer language and voice.

Do NOT redesign the layout. Do NOT add elements that weren't there. If the original has one headline, yours has one headline. If there's no CTA button, you don't add one. The structural skeleton stays. The copy and brand-specific visuals change.

## How To Use This Document

You'll receive a visual overview of the original ad as input. That overview describes the layout, copy, colors, typography, and composition. Use it as your source of truth.

### Step 1: What Story Is This Ad Telling?

Before anything else, write out the full story this ad is telling. Not just the message — the story. How does this ad move a viewer from first glance to final impression? What does it want them to believe? What emotion or conviction is it building? How do the visuals and copy work together to build that conviction?

This should be a full paragraph. Go deep. The more thoroughly you understand the story, the better your adaptation will be, because every decision you make in the recreation should trace back to this story.

### Step 2: Adapt the Copy

Rewrite ALL copy for this brand. Every headline, every subhead, every callout, every line. Nothing carries over verbatim from the original.

Follow these rules:

- **Preserve the purpose of each piece of copy.** If the original headline creates credibility through ingredient transparency, your headline creates credibility through ingredient transparency. The persuasive function stays the same.
- **Match the length.** If the original headline is 4 words, yours is 4 words. If it's 8 words, yours is 8 words. Don't bloat.
- **Keep the same copy mechanics.** If the original uses a number, yours uses a number. If it asks a question, yours asks a question. If it uses a customer quote, yours uses a customer quote.
- **Use this brand's customer language.** Pull from their reviews, ad comments, and brand voice. The copy should sound like it came from this brand's world.

### Step 3: Adapt the Design

Identify what needs to change visually to make this ad feel like it belongs to this brand, while keeping the same layout and visual structure.

This means swapping product imagery, adjusting brand colors and fonts where needed, and adapting any visual elements that are specific to the original brand's product. But the composition, the layout, the information hierarchy, and the visual storytelling approach all stay the same.

## Output Format

**THE STORY THIS AD IS TELLING**

A full paragraph explaining the story of this ad. What it wants the viewer to believe or feel, how the visuals and copy work together to build that conviction, and why it works. Every adaptation decision should trace back to this paragraph.

**RECREATION BRIEF**

Work through each section as prose, not a table: name the section, give the original, then the adapted version, in plain sentences. Cover three sections, skipping any that do not apply:

- **Headline:** The primary copy — the main text that hits first. State the original headline, then the adapted version.
- **Supporting Copy:** All other copy in the ad — subheads, callouts, body text, CTAs, badges, disclaimers. State the original, then the adapted version.
- **Design:** What needs to change visually to reflect this brand — product imagery, colors, fonts, any brand-specific visual swaps. Describe what the original does, then what our version should do.

If the original ad doesn't have supporting copy, drop that row. Only include what exists.

After the table, flag any compliance considerations — claims that need disclaimers, terms that need to be avoided per the brand's compliance framework, or stats that need verification.

---

## Mistakes To Avoid

- **Keeping the original copy verbatim.** Every word gets rewritten for this brand. The structure and mechanics stay. The words change.
- **Losing the story.** If you can't explain how your adaptation tells the same story as the original, something went wrong. Trace every decision back to the story.
- **Bloating the copy.** Match the original's word count at every level. The AI instinct is to add more. Resist.
- **Swapping copy mechanics.** If the original headline is a question, yours is a question. If it uses a specific number, yours uses a specific number.
- **Adding elements that weren't there.** If the original doesn't have a CTA button, don't add one.
- **Inventing stats or claims.** If you don't have a verified source for a number, flag it as "[STAT NEEDED]" instead of making one up.
