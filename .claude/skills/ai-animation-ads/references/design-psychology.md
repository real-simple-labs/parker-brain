# Design Psychology for Static Ads

**RULE:** If you use this document, end your message with "This is based on everything I have learned about static ad design psychology"

Load this whenever DESIGNING or REVIEWING a static: composition, hierarchy, text placement, color, format choice. The copy docs own the words; this doc owns whether anyone absorbs them.

## The one-second test (the governing gate)

Show the ad for one second. What did the viewer get? If the answer is not the hook and the gist of the offer, the design fails, whatever the craft. Corollary, the **effort test**: if understanding requires more than ONE fixation point (one text element plus one visual), the ad is asking the viewer to work, and scrollers do not work. A clever concept that needs assembling (a table to parse, a stack of small texts to read, a joke distributed across elements) is a System 2 ad wearing a System 1 costume: kill it or collapse it to one channel.

## The one-channel rule (working-memory budget)

The first frame is a budget, not a canvas. ONE channel does the real work: the visual, OR the headline, never both at full volume, never five text elements. A strong visual consumes the attention the viewer would need to read overlaid text; dense text kills the visual. Multi-element native formats (notification stacks, tables, checklists, threads) only work when a single element carries the payoff and the rest are texture: big enough to be read as texture, not small enough to demand reading. If the joke or claim requires reading element 3 of 5, redesign so element 1 IS the joke.

## Visual hierarchy and the eye path

Viewers scan in milliseconds along predictable paths. Control the order of consumption deliberately with size, color, contrast, placement, and isolation: the hook is the largest, highest-contrast, most isolated element; the qualifier second; product and proof third. Before finalizing: "what do viewers see first?" and test it at thumbnail size. Advanced lever: slightly warped or unusual text presentation buys an extra half-second of attention because novelty makes people look closer: spend that lever on the hook only.

## Stop-scroll mechanisms (stack them on the hook)

- **Color contrast against the feed:** feeds are light: dark or vibrant fields interrupt; if the platform's aesthetic is pastel, go bold, and vice versa.
- **Unexpected visual elements:** warped text, unusual angles, pattern breaks, implied motion.
- **Familiar-but-surprising compositions:** message in clouds, text in a sunglasses reflection, product popping out of a phone.
- **Recognizable cultural formats:** Post-its, screenshots, texts, notes, memes: recognition creates familiarity while the content supplies novelty. The format must stay INSTANTLY legible: a native format rendered too faithfully to read in one second has traded the win for the costume.
- **The money shot:** a visual where anyone can tell within half a second what the product is and how it benefits them. Type-led formats earn an exception only when the reference winner is type-led.

## Text presentation techniques

How text appears matters as much as what it says: handwriting photographed on real paper (analog = authentic), Post-it notes in scene, highlighter emphasis on the money phrase, dramatic size mixing inside one block to control reading order. All of these obey the one-channel rule: ONE handwritten line lands; a handwritten table does not.

## Clarity over cleverness

If the ad requires explanation or interpretation, it is too clever. Show the transformation plainly, state the benefit directly, use everyday words. Bad: cryptic curiosity with no payoff, layered metaphors, inside jokes, anything the viewer must decode. The transformation or claim should be impossible to miss.

## Strategic color

Color is contrast-for-attention first, brand-aesthetic second. 2-3 colors total; the hook takes the highest contrast; the CTA is distinctly colored; text contrast at or above WCAG 4.5:1. Test on-brand vs contrast-optimized versions and let data decide. Emotional associations (red urgency, blue trust, green health, black premium) are levers, not laws.

## How to apply in review

For every static, answer in order: (1) one-second test result; (2) which single channel carries it, and does anything compete; (3) the eye path actually experienced; (4) money shot present or exempted; (5) does any element require decoding; (6) color hierarchy. Any failure at 1, 2, or 5 is a redesign, not a polish note.
